Ініціалізація проекту
   git init

Клонування репозиторію
   git clone адрес репозиторію

Створити нову гілку
   git checkout -b назва гілки

   -b означає створиться нова гілка 

Для переходу на іншу гілку
    git checkout назва гілки

    git checkout -  // перехід на будь-яку попередню гілку

Додати всі файли до коміту
   git add .

   git add назва файлу  //додасть певний файл 

Для перевірки статусу
   git status

Закомітити, зберегти зміни. Зміни будуть збережені на вибраній гілці та локально
   git commit -m 'повідомлення'

Додати зміни на віддалений репозиторій
   git push

Додати гілку на віддалений репозиторій та зберегти зміни
   git push -u назва гілки

 Для того щоб додати зміни в основну гілку на віддаленому репозиторії
 зайти в github та зробити pull request

 Щоб ці зміни з'явилися на основній гілці локально 
   gitt pull

Для накладання змін з однієї гілки на другу
   git merge назва гілки

   git merge -  // внесе зміни з попередньої гілки
